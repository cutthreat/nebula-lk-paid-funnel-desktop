# Nebula LK designer web panel

Внешняя веб-панель по личному кабинету Nebula.

Публикуется через GitHub Pages.

Local entry:

```text
index.html
```

Included:

1. `index.html` - главная панель.
2. `onboarding-modal.html` - отдельный интерактивный onboarding modal, как в исходном локальном наброске.
3. `matching-payment-example.html` - интерактивный пример пути к консультации: рекомендованный эксперт, запасные варианты и офер/оплата.
4. `paid-funnel.html` - полный кликабельный десктоп макет платной консультации.
5. `chat-paid-session-mockups.html` - кликабельные десктоп-состояния платного чата.
6. `global-fallback-lab.html` - глобальная интерактивная карта fallback-состояний всей paid-воронки.
7. `chat-designer-minimum.html` - сжатый бриф по состояниям платной услуги чата.

Фокус панели: десктоп. В `onboarding-modal.html` сохранен старый мобильный блок как справочный материал из исходного макета.
