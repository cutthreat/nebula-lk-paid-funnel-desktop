Что реально дорисовать для чата

Ниже только состояния, без которых платная услуга может сломаться при запуске. Базовый desktop chat уже есть.

Открыть бриф по чату
1. Офер / оплата / creditsОкно офера, покупки кредитов, top-up, success/fail/pending.
2. Статус платной сессииLive/waiting/ended, стоимость, остаток времени или кредитов.
3. Вопрос перед чатомВопрос клиента и контекст до старта консультации.
4. Обрыв связиПлохая связь эксперта/клиента, reconnect, billing paused, fallback.
5. Эксперт недоступенOffline/busy/delayed, wait/async/another expert.
6. Low/no creditsПредупреждение, остановка сессии, top-up, save transcript.
7. Failed send/uploadНе отправилось сообщение/вложение, retry/remove.
8. Ended transcriptИтог, сохранение, follow-up, rating.
9. Support/refund/reportЖалоба, refund, support request, статусы обращения.
