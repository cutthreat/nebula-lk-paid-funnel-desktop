Новый интерактивный проход по всей воронке: неполные данные онбординга, слабый матчинг, занятый эксперт, ошибка оплаты, free→paid, подключение без списания, live-сессия, credits, связь, отправка, завершение и поддержка.

Что реально дорисовать для чата

Ниже только состояния, без которых платная услуга может сломаться при запуске. Базовый десктоп-чат уже есть.

Открыть бриф по чату
1. Офер / оплата / кредитыОкно офера, покупки кредитов, пополнение, успех/ошибка/ожидание.
2. Статус платной сессииlive-чат / ожидание / завершено, стоимость, остаток времени или кредитов.
3. Вопрос перед чатомВопрос клиента и контекст до старта консультации.
4. Обрыв связиПлохая связь эксперта/клиента, переподключение, списание на паузе, запасной путь.
5. Эксперт недоступенне в сети / занята / задержка, ждать/async/другой эксперт.
6. Мало / нет кредитовПредупреждение, остановка сессии, пополнение, сохранение стенограммы.
7. Ошибка отправки / загрузкиНе отправилось сообщение/вложение, retry/remove.
8. Завершенная стенограммаИтог, сохранение, следующий вопрос, оценка.
9. Поддержка / возврат / жалобаЖалоба, refund, запрос в поддержку, статусы обращения.
